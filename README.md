# Customer Churn Analysis - Telecom Industry
 Overview
This project presents an end-to-end Customer Churn Analysis for a telecom company using real-world retention data.
The goal is to:
#Measure the scale of customer churn

Identify high-risk customer segments

Quantify revenue at risk due to churn

Analyze behavioral, service-related, and demographic drivers

Provide actionable insights for retention and customer experience improvement
